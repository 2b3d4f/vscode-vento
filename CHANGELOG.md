# Change Log
All notable changes to the "vento" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how
to structure this file.

## [0.0.6] - 2024-03-19
### Fixed
- Missing template keywords [#3].

## [0.0.5] - 2023-09-16
### Added
- Support for front matter [#2].
- Support for new keywords `layout`, `echo`.
- Snippets for the new features (`layout`, `echo`, `function`, `import`, `export`).

## [0.0.4] - 2023-07-16
### Added
- Emmet support [#1]

## [0.0.3] - 2023-07-03
### Fixed
- HTML syntax highlight.

## [0.0.2] - 2023-06-14
### Added
- Icon to `package.json`.
- Screenshot to `README.md`.

### Fixed
- `for key,value` snippet.

## [0.0.1] - 2023-06-13
Initial release
